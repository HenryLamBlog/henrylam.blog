StyleMap HK: Personalized Clothing Style and Fashion Assistant

In Hong Kong's densely packed urban environment, finding clothes that match both style preferences and fit requirements can be challenging. StyleMap HK addresses this problem by developing an integrated solution that combines personalized clothing recommendations based on individual measurements and style preferences with local store mapping functionality to guide users to physical retail locations where suitable items can be found.

StyleMap HK mobile application interface showing personalized recommendations

Problem Definition

Despite the rise of online shopping, many consumers still prefer to try on clothes before purchasing, especially when concerned about fit and style. However, navigating Hong Kong's complex retail landscape can be overwhelming. StyleMap HK bridges this gap by combining personalized styling recommendations with geolocation services to create a seamless shopping experience.

Methodology

Our approach integrates multiple data collection and processing methods:

User Profile Creation

Collection of body measurements, personal style preferences, demographic data, and shopping habits

Store Mapping

Comprehensive database of Hong Kong clothing retailers with detailed inventory categorization

Image Recognition

Machine learning algorithms to analyze clothing attributes from visual data

Recommendation System

AI-driven matching algorithm connecting user profiles with appropriate clothing items and stores

System Architecture

StyleMap HK features a three-tier architecture:

  • Mobile application built with React Native for cross-platform compatibility
  • User-friendly interface for profile creation, clothing browsing, and store navigation
  • AR components for virtual try-on capabilities using smartphone cameras
  • Interactive maps showing retail locations with real-time data

  • Recommendation engine using collaborative and content-based filtering
  • Image processing module for garment recognition and classification
  • Geolocation services for proximity-based store recommendations
  • Route optimization algorithms for efficient shopping itineraries

  • User profile database storing measurements, preferences, and history
  • Clothing database with detailed attribute categorization
  • Store inventory database updated through retail partnerships
  • Spatial database for physical store locations and mapping

User Profiling System

The user profile creation includes:

  • Body Measurements: Comprehensive collection of up to 15 key measurements
  • Style Preference Assessment: Interactive questionnaire and image selection to determine style preferences
  • Occasion-Based Recommendations: Customized suggestions based on specific events or needs
  • Budget Considerations: Price range filtering for personalized recommendations

Image Recognition System

Our image recognition technology:

  • Uses deep learning models (CNN architecture) trained on clothing datasets
  • Identifies key clothing attributes including color, pattern, style, fabric, and garment type
  • Processes both user-uploaded images and store inventory photos
  • Enables similarity matching between user preferences and available inventory

Recommendation Algorithm

The recommendation system employs:

  • Hybrid Filtering: Combination of collaborative and content-based approaches
  • Body-Type Matching: Algorithms matching clothing proportions to user measurements
  • Style Consistency: Ensuring recommendations align with expressed style preferences
  • Contextual Awareness: Considering location, season, and social context in suggestions
  • Feedback Learning: Continuous improvement based on user interactions and purchases

Store Mapping Features

The geolocation component includes:

  • Comprehensive database of Hong Kong clothing retailers across all districts
  • Integration with public transportation data for convenient navigation
  • Real-time inventory checking through retail partnerships
  • Optimized shopping routes based on user preferences and store proximity
  • Price comparison across multiple retail locations
